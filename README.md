# 基于GO标准库改造的证书生成工具包

## 最新更新（v0.4.0）

1. 基于golang 1.24.4版本标准库重构，跟进官方的优化；

2. 避免了对ecdsa包的入侵，使项目更加简洁；
3. 对所有改动进行标柱，便于学习和改进；
4. 根据实际使用场景，优化了对外暴露的工具函数；
5. 优化了很多内部逻辑；

## 标柱说明

- 标准库原来的代码使用`// Original`进行标柱，可全局搜索查看；

- 增加的代码使用`// Modify-----------`进行标柱，可全局查看；
